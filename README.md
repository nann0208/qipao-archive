# 民国海派旗袍史料库（网站版）

一个**纯前端、零安装、双击即用**的史料管理工具，专为设计史研究打造。

## 快速开始

**双击 `index.html`** 即可在浏览器中打开。推荐使用 Microsoft Edge 或 Google Chrome。

## 功能特性

- 🎴 **彩色卡片浏览** —— 按主题议题色彩区分（旗袍款式胭脂红、性别议题玫瑰粉、外贸藏青等）
- 🔍 **全文检索** —— 标题、内容、个人分析、关键词均可搜索
- 🏷️ **多维筛选** —— 按类型（报刊/专著/档案/图像）+ 议题分类组合筛选
- 📄 **PDF 右侧分屏** —— 详情页左侧信息、右侧自动加载原文
- 🖼️ **图像预览** —— 直接显示 jpg/png 等图像
- ✏️ **网页表单录入** —— 随时添加新史料，无需打开 Excel
- 💾 **本地数据存储** —— 所有数据保存在浏览器 localStorage 中
- ⬇️ **一键导出** —— 数据可导出为 data.js 文件长期备份

## 数据管理

### 添加/编辑/删除
所有变更保存在浏览器的 localStorage 中（仅本机有效）。

### 数据备份与同步

**重要！** 务必定期备份数据：

1. 在主页点击右上角「⬇ 导出数据」
2. 浏览器会下载 `data-YYYY-MM-DD.js` 文件
3. 将该文件移入 `data/` 文件夹，重命名为 `data.js`（覆盖原文件）
4. 完成。下次打开时，本地修改将作为新的"初始数据"加载

不备份的风险：浏览器清除缓存、换浏览器、换电脑等场景下数据会丢失。

## 字段说明

| 字段 | 说明 |
|------|------|
| 史料编号 | 自动生成，格式 SL-YYYY-NNN |
| 史料类型 | 报刊文章 / 专著 / 档案文件 / 图像 |

| 研究议题 | 多选，决定卡片颜色 |
| 重要程度 | ⭐⭐⭐ 核心 / ⭐⭐ 参考 / ⭐ 备用 |
| 核心内容 | 史料摘录或概述 |
| 个人分析 | 你对这条史料的研究判断 |
| 引用片段 | 可直接用于论文的原文 |
| 关键词 | 自由标签，用于检索 |
| 关联原文 | 文件相对路径（PDF/JPG/DOCX） |

## 关联原文档案

文件路径采用相对路径：
- 网站位置：`D:\AAA好好读书\aaa史料收集\shiliao_website\index.html`
- 原文位置：`D:\AAA好好读书\aaa史料收集\《申报》\xxx.pdf`
- 相对路径：`../《申报》/xxx.pdf`

初始 142 条记录已自动匹配相应文件路径。新添加的记录请手动填写。

## 已知限制

1. **`.docx` 文件无法在浏览器内预览**：会显示「在系统中打开」按钮调用本地 Word
2. **localStorage 容量限制**：约 5-10MB，对纯文本元数据足够（不存图片本身）
3. **跨设备使用**：当前为本地方案。如需多设备同步，可将整个文件夹放入云盘

## 开发者信息

技术栈：纯 HTML + CSS + 原生 JavaScript（无任何依赖、无构建步骤）

如需修改：
- 颜色：编辑 `assets/style.css` 顶部的 `:root` 变量和 `assets/storage.js` 的 `TOPIC_COLORS`
- 字段：同时修改 `assets/storage.js`、`assets/add.js`、`assets/detail.js`
- 议题列表：修改 `assets/storage.js` 的 `ALL_TOPICS`
